## 1. 产品概述

家族树管理APP是一款帮助用户创建和管理家谱的应用，支持添加家庭成员信息（姓名、出生日期、逝世日期、关系类型），自动校验关系逻辑，实时生成可交互谱系图。同时提供家庭活动管理、生日提醒、家族编年史等丰富功能，让家族记忆得以传承。

- 目标用户：希望记录和传承家族历史的家庭用户
- 核心价值：通过数字化手段保存家族信息，智能提醒促进家庭互动，编年史功能让家族故事永续流传

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家族管理员 | 邀请注册 | 创建家谱、管理成员、发布活动、管理编年史 |
| 家族成员 | 邀请注册 | 查看家谱、报名活动、上传照片和故事 |

### 2.2 功能模块

1. **仪表盘页面**：家族概览、近期活动、生日提醒、快捷操作入口
2. **家谱树页面**：可交互谱系图、成员添加/编辑、关系校验提示
3. **家庭活动页面**：活动列表、活动发布、报名统计、提醒设置
4. **家族编年史页面**：时间线视图、照片和故事上传、PDF导出

## 3. 核心流程

### 3.1 添加家庭成员流程

用户点击"添加成员"→ 填写姓名（必填）、出生日期（必填，校验格式）、逝世日期（选填，校验格式）、关系类型（必填）→ 选择关联成员 → 系统校验关系逻辑（如父母子女年龄差≥15岁）→ 校验通过则添加成功并更新谱系图 → 校验不通过则弹出明确提示

### 3.2 活动管理流程

管理员发布活动（标题、时间、地点、描述）→ 成员浏览活动列表 → 成员点击报名 → 系统自动统计报名人数 → 活动前一天系统推送提醒通知

### 3.3 生日提醒流程

系统每月1日自动扫描全部成员生日 → 筛选未来7天内过生日的成员 → 推送祝福提醒和礼品建议

### 3.4 编年史生成流程

成员上传照片/故事（填写日期和描述）→ 系统按时间线自动归集 → 生成家族编年史视图 → 用户可导出为PDF存档

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深棕暖色（#5C3D2E）+ 金色点缀（#C9A96E），传达家族传承的温暖与厚重
- **辅助色**：米白色（#F5F0EB）背景，深灰（#2D2D2D）文字
- **按钮风格**：圆角矩形，3D 微凸效果，悬停时加深阴影
- **字体**：标题使用 "Noto Serif SC" 衬线体，正文使用 "Noto Sans SC" 无衬线体
- **布局风格**：左侧导航栏 + 右侧内容区，卡片式布局
- **图标风格**：线性图标，搭配 Lucide React 图标库
- **装饰元素**：族谱纹样、古铜色边框、宣纸质感的背景纹理

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|

### 4.3 响应式设计

- 桌面优先设计，最小宽度1024px
- 平板适配：侧边导航收起为图标模式
- 移动端适配：底部标签栏导航，卡片单列布局，谱系图支持触摸缩放

### 4.4 动效设计

- 页面切换：淡入淡出 + 微微上滑
- 卡片悬停：阴影加深 + 微微上浮
- 谱系图节点点击：脉冲动画 + 信息面板滑出
- 时间线滚动：视差效果 + 逐帧渐入
- 表单校验：红色边框抖动 + 错误文字渐入
